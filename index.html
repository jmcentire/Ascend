ascend

AI-powered engineering management CLI. Track your team, analyze performance, prepare for 1:1s, and detect risks — all from the terminal.

$ pip install -e .
ascend tui

Features

Everything you need to lead engineering teams effectively.

📋

Roster Management

Track team members with GitHub, Slack, email, title. Hierarchical teams, bulk CSV import, searchable flags (oncall, PTO, PIP).

🎙

Meeting Intelligence

Ingest transcripts, get AI summaries, extract action items and decisions. Full-text search across all meetings.

📈

Performance Tracking

Aggregate GitHub commits, Linear issues, Slack activity into daily snapshots. Track velocity trends over 30+ days.

⚠

Risk Detection

Algorithmic detection of flight risk, burnout signals, bus factor, and underperformance. Early warning before it's too late.

🧑‍🏫

AI Coaching

STAR behavioral assessments, personalized coaching suggestions, and AI-generated 1:1 prep plans with full context.

📊

Reports

Performance metrics, team health, project progress, git analytics, org dashboard. Free-form AI reports on any question.

🎯

Goals & Planning

OKR management, performance improvement plans, career development tracking. Full planning cycle support.

⏰

Scheduling

Automated task scheduling with cron expressions. macOS launchd integration, presets for daily, weekly, monthly.

🖥

Interactive TUI

Full terminal interface with mouse support, keyboard nav, command palette. Six screens: Dashboard, Roster, Meetings, Reports, Coaching, Schedules.

Quick Start

Up and running in under a minute.

Install

$ git clone https://github.com/jmcentire/Ascend.git
$ cd Ascend
$ pip install -e .
$ ascend init

Configure

Set your API keys for AI features and integrations.

# Required for AI features (summaries, coaching)
$ export ASCEND_ANTHROPIC_API_KEY="sk-ant-..."

# Optional integrations
$ export LINEAR_API_KEY="lin_api_..."
$ export SLACK_BOT_TOKEN="xoxb-..."

$ ascend config set github_org your-org

Import your team

# Bulk import from CSV
$ ascend roster import team.csv

# Or add one by one
$ ascend roster add "Alice Smith" \
    --email alice@co.com --github alicesmith --title "Senior Engineer"

Sync & analyze

$ ascend sync                # Pull from GitHub, Linear, Slack
$ ascend report dashboard    # Org-wide summary
$ ascend coach risks         # Risk detection dashboard
$ ascend tui                 # Launch interactive UI
